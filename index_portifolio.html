Projetos desenvolvidos, para saber mais acesse minhas redes.

Poli Wear é um site desenvolvido para simplificar a compra de camisetas escolares e de cursos, substituindo o processo manual de pagamento e evitando atrasos. Ele foi projetado para resolver um problema real dentro do ambiente escolar, trazendo praticidade e organização.

O ADM Soluções é uma aplicação web cliente-servidor desenvolvida para automatizar e gerenciar o ciclo completo de agendamentos realizados por um prestador de serviços. O projeto foi idealizado para substituir a gestão manual, fornecendo uma interface intuitiva.

O projeto Câmbio Fácil foi pensado para auxiliar usuários que precisam realizar conversões de valores entre diferentes moedas de forma rápida, prática e confiável. O projeto visa entregar uma alternativa simples e eficiente e que garanta acesso a informações atualizadas de câmbio.
